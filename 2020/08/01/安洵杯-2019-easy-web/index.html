Web狗如何在险恶的CTF世界中存活？–怎么可能存活，想多了

考点：编码、md5碰撞、RCE绕过

拿到题目后load一下url，发现有两个参数img=TXpVek5UTTFNbVUzTURabE5qYz0&cmd=。img参数看着像base64解码后得到MzUzNTM1MmU3MDZlNjc=，再次解码3535352e706e67，用十六进制转ascii得到555.png。网页源码里有图片的base64的值，于是尝试将index.php进行hex+base64+base64得到TmprMlJUWTBOalUzT0RKRk56QTJPRGN3，传入其传入img得到base64编码后的网页源码，解码后得：

php

(preg_match("/flag/i", $file))这个把参数里的flag过滤掉了，我们就没法就去读flag文件了，所以利用下面的echo '$cmd';来进行命令执行，在此之前有一个md5强碰撞

Code
php

禁用了很多读取文件常用的命令，比如tac、nl、more、head、tail、cat、echo、vi等被ban了，单引号、@、分号、逗号、$、{、}等构造符号也被禁了

dir没被ban，但是\xA0被ban了，于是用dir%20/看一下根目录都有啥文件，发现flag文件

1.利用ca\t或者/bin/c\at来绕过，利用反斜杠产生转义字符绕过后仍可执行

2.sort命令：sort将文件的每一行作为一个单位，相互比较，比较原则是从首字符向后，依次按ASCII码值进行比较，最后将他们按升序输出。
