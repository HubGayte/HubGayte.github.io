由于上周周四、周五、周六玩嗨了，同时以及认为自己可以1秒5道编程，导致周日混天儿的同时去做了做防灾科技学院的GKCTF。

做的时候misc最后一个闪瞎眼、猜长短的题没做出来，web照样一道不会（骄傲个屁，跟前几天BJD 3rd安恒月赛似的，等环境再上了就去复盘一下），密码也做出来两道水题，这随便记录一下吧。

Misc

问卷调查

可惜了没进前六、也没进前三十，要是最后那道misc出了或者出一道web就进前三十了呀。。（别说了，还是菜

Pokémon

很有趣的一道题，让我这个没玩过gba游戏的孩子也拿模拟器玩了一把

整个模拟器（我用的VisualBoyAdvance1.8CN），进游戏认识下键位，跟着剧情走就完事了，到103号路上会发现flag被拿树拼了起来

code obfuscation

考点：二维码修复、文件提取、base58、js反混淆

这二维码看到我是不想修它的，因为我不会ps。。幸亏是个21*21，然后我就在QRazyBox修了一下。（听说拿ps斜变换，然后把空隙填充一下就能扫出来了）

扫码得到：base(gkctf)

图片放到010里看一下，发现结尾后面还有个rar，提取出来（然后我就拿上面的密码解了半天，还以为要修复压缩包了。。其实我也想过base加密之后是密码，但是就试了个64

之后题目提示出来了：压缩包密码是加密过的，然后我就16、32、36、58这样往上试，到58成功了

根据题目名称知道考得代码混淆，搜一个反混淆网站，解码得到：

Harley Quinn

考点：电话音、九宫格拼音、FreeFileCamouflage隐写

给了个压缩包注释里有FreeFileCamouflage（当时不知道这个软件，把它放进百度翻译里面去了。。啥信息也没得到

解压后给了一张图片、一个音频，先看一下图片，图片尾有一堆像base64的东西，解码啥也没得到

然后看音频，尾部有拨号音，手解一下（不习惯用工具。。（这个自从我知道这个知识点后，从xctf开始见了好多次了。。感觉这没啥考头，就考个新颖，但是大家基本都知道了）

得到：#222833344477773338866#

提示九宫格，根据手机按键解码得到：ctfisfun

然后根据提示：FreeFileCamouflage，下载的文件可能显示乱码，才让我想到这可能是个软件。

把jpg放进去，输入解出的字符串

解密得到文本，里面就是flag：flag{Pudd1n!!_y0u_F1nd_m3!}

Sail a boat down the river

考点：观察能力、数独、AES

这题给👴看傻了。。

拿到题目，给了个视频，里面有张二维码，扫码得到一个百度网盘的网址，需要提取码

然后开始分析mp4里有啥隐写方式，拿exiftool和MediaInfo瞅了瞅啥也没发现

然后看提示说：闪烁的光芒，发现视频里有个刷卡通行机有闪烁频率（我第一次看以为是车通过的时候触发的），然后我0.2倍速慢放得到闪烁频率4353（感觉是摩斯密码，拿去按照..../.../...../...解码得到HS5S，大小写都试了不是提取码）我又看了几遍应该是这么闪的，但就是解不出来，然后以为是视频背景的明暗变化（有、扯淡了）然后就放弃了

看官方wp里面说，这玩意还能看出来长短（我人傻了。。）

视频118-130 200-208 320-334 410-418帧有明显闪烁，可以看出是morse密码

pr或ps在这⼏个⻓间隔闪光区间逐帧看

短是1帧，⻓是3帧，拼合得到 -.–/.–/—../–. 解码得到yw8g

提取下来是个数独

Code

找个求解器在线求一下（想玩、不会玩23333

根据提示第一列九宫格从左到右从上到下后来改成第一行九宫格从左到右从上到下

得到密钥：52693795149137

常见的对称加密算法有DES、3DES、AES、RC5、RC6。试试就完事了

然后发现AES、输出位hex时可以成功解密

拿去解密压缩包，得到逆光 vocal.ovex

下载个overture写谱软件，打开得到flag

CheckIN

上来直接给源码

php

意思就是传入base64加密过的Ginkgo会进行作为eval的参数执行(也就是传个php语句(没被ban的))

于是传个一句话eval($_POST[pass]);—base64—>ZXZhbCgkX1BPU1RbcGFzc10pOw==

连上后发现根目录flag文件里没有flag，有个readflag，里面是乱码，但是有cat /flag，应该是个执行文件，在蚁剑终端执行一下，返回ret=127，应该是system被禁了

拿插件看一下phpinfo（还发现了flag_not_here，哈哈哈）

于是用bypass PHP7.0-7.3 disable_function的PoC打一下

https://github.com/mm0r1/exploits/blob/master/php7-gc-bypass/exploit.php

要把里面的执行命令改成/readflag，拿蚁剑上传一下（但是html/www目录没有权限。。

/tmp和 /var/tmp目录供所有用户暂时存取文件，即每位用户皆拥有完整的权限进入该权目录，去浏览、删除和移动文件。

上传到/tmp目录下面，然后再用include包含（也可以用别的没被ban的）一下这个poc，得到flag

cve版签到

考点：cve-2020-7066、ssrf

在PHP版本7.2.x、7.2.29以下、7.3.x低于7.3.16和7.4.x在7.4.4以下，在使用用户提供的URL的get headers()时，如果URL包含0(\0)字符，URL将被静默截断。这可能导致一些软件对get headers()的目标做出错误的假设，并可能将一些信息发送到错误的服务器。

看到一个链接view ctfhub，点一下会跳转，源码里有Venom 在线招人（我太菜了

发现/?url=http://www.ctfhub.com，跳转后发现Headers中存在Hint:Flag in localhost

因为它前面提到访问的是*.ctfhub.com，因此将www替换为127.0.0.1并用%00截断，得到下面提示

Code

因此将127.0.0.1替换为127.0.0.123，访问即可得到flag

老八小超市儿

打开后是个shopxo界面，在网上搜一下shopxo的漏洞

发现默认后台/admin.php有弱口令账号：admin 密码：shopxo

登录成功后在应用中心-应用商店-主题，然后下载默认主题，下载下来把你的shell放到压缩包的default\_static_ 目录下

回到网页上，找到网站管理-主题管理-主题安装，然后选择你加入shell后的主题压缩包进行上传

安装成功后，拿蚁剑连一下地址是：xxxx/public/static/index/default/7788.phtml
